## باشە! بەواری تەواو ڕوونت دەکەمەوە

---

## سیستەمی نەخۆشخانەی گەورە — Architecture تەواو

---

### 🏗️ مۆدیوولەکان (Modules)

---

#### 1️⃣ OPD — نەخۆشی دەرەوە (Outpatient Department)
- تۆمارکردنی نەخۆشی تازە
- دانانی کاتی پزیشکی
- پشکنینی پزیشک + نووسینی تەشخیص
- نووسینی ڕێنووسی دەرمان (Prescription)
- ئیحالەدان بۆ لابراتوار یان دەستەی تایبەت

---

#### 2️⃣ IPD — نەخۆشی ژوورەوە (Inpatient Department)
- خەوەندنی نەخۆش بە جێگا (Admission)
- دیاریکردنی ژوور و بەستەر
- شوێنکەوتنی ڕۆژانەی پزیشک (Rounds)
- مامەڵەکردن و دەرکردنی نەخۆش (Discharge)
- بەرپرسایەتی پەرستار بەسەر نەخۆشەکان

---

#### 3️⃣ Emergency — ئەوارگە
- تۆمارکردنی خێرا (Fast Registration)
- Triage — پلەبەندی گەورەیی نەخۆشی
- شوێنکەوتنی کاتی ئەوارگە
- گواستنەوەی ئارام بۆ IPD یان OPD

---

#### 4️⃣ Surgery — نەشتەرگەری
- دانانی کاتی نەشتەرگەری
- شوێنکەوتنی ژووری نەشتەرگەری (OT)
- تیمی جراحی + ئانستیزیا
- ڕاپۆرتی نەشتەرگەری

---

#### 5️⃣ Laboratory — لابراتوار
- داوای تاقیکردنەوە لە پزیشکەوە
- تۆمارکردنی ئەنجام
- چاپکردن/ناردنی ڕاپۆرت بۆ نەخۆش
- شوێنکەوتنی نموونەکان

---

#### 6️⃣ Radiology — تەصویر
- داوای ئێکس-ڕەی، MRI، CT Scan
- بارکردنی وێنەکان
- ڕاپۆرتی رادیۆلۆجیست

---

#### 7️⃣ Pharmacy — دەرمانخانە
- وەرگرتنی ڕێنووسی دەرمان
- فرۆشتنی دەرمان بە نەخۆش
- بەڕێوەبردنی کۆگا (Stock)
- ئاگادارکردنەوەی کەمی کۆگا
- بەرواری بەسەرچوون (Expiry)

---

#### 8️⃣ HR & Payroll — کارمەند و مووچە
- تۆمارکردنی پزیشک، پەرستار، کارمەند
- شیفتەکان و خەڵەتی
- مووچەی مانگانە
- مۆڵەتی نەخۆشی و ئاسایی

---

#### 9️⃣ Billing & Finance — داهات و داڕاو
- بەرپرسایەتی ژمارەکانی نەخۆش
- پارەدانی کاش، کارت، بیمە
- فاتووراکان
- ڕاپۆرتی داهات ڕۆژانە/مانگانە

---

#### 🔟 Inventory — کۆگای گشتی
- ئامرازی پزیشکی
- بەڕێوەبردنی کڕین و دابەشکردن
- تۆمارکردنی تەواو و خراپبوون

---

#### 1️⃣1️⃣ Dashboard & Reports — داشبۆرد
- ئامارە ڕۆژانەکان
- گرافی نەخۆشەکان
- ڕاپۆرتی گشتی بۆ بەڕێوەبەری نەخۆشخانە

---

#### 1️⃣2️⃣ Login & Roles — ئەندازیاری دەسەڵات
| ڕۆڵ | دەسەڵات |
|---|---|
| Admin | هەموو شت |
| پزیشک | نەخۆشی خۆی، ڕێنووس، تەشخیص |
| پەرستار | IPD، ئەوارگە، بەسترەکان |
| دەرمانخانەچی | Pharmacy تەنها |
| کاشێر | Billing تەنها |
| لابراتوار | Lab تەنها |
| بەڕێوەبەر | داشبۆرد و ڕاپۆرت |

---

### 💻 Technology Stack

| بەش | تەکنەلۆژیا |
|---|---|
| Frontend | React + Tailwind CSS |
| Backend | Python / Node.js |
| Database | PostgreSQL |
| Auth | JWT + Role-based |
| Print | PDF Export |
| Charts | Recharts |

---

### ❓ ئێستا پێم بڵێ:

**١. لە کوێوە دەست پێ بکەم؟**
- داشبۆرد + تۆمارکردنی نەخۆش (OPD) — باشترین دەستپێک
- یان مۆدیولێکی تایبەت دەتەوێت پێشتر؟

**٢. زمانی پرۆگرامنووسی:**
- React (JSX) بۆ UI
- یان دەتەوێت Backend (Python/Node) لەگەڵدا بنووسم؟

**٣. زمانی ئینتەرفەیس:**
- کوردی تەنها؟
- یان کوردی + عەرەبی + ئینگلیزی؟
